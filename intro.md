# Геоаналитика на Python

Интерактивное пособие, посвящённое **основам работы с пространственными данными на Python**.

На платформе [Boosty](https://boosty.to/bellamir) раз в неделю будут выходить видеоразборы занятий, а также публиковаться дополнительная полезные материалы.

## Курс-проект

Шаг за шагом вы будете **осваивать методы пространственного анализа** и знакомиться с библиотеками Python для работы с геоданными, одновременно **выполняя проект**. На конкретном примере вы сможете оценить доступность объектов социальной инфраструктуры и отобразите результаты на интерактивной карте.

Курс состоит из модулей. Каждый модуль посвящён отдельной теме и включает задание — один из этапов итогового проекта.

## Основные темы

В рамках курса мы рассмотрим:

- Форматы пространственных данных
- Проекции и системы координат
- Пространственные операции
- Основы сетевого анализа
- Растровые данные и зональную статистику
- Создание интерактивных карт

## Формат

Для начала достаточно базового понимания Python.

Курс состоит из набора Jupyter-ноутбуков, объединённых в Jupyter Book.

Модули можно проходить как по отдельности, так и последовательно.

Каждый модуль включает:

- краткую теорию,
- примеры кода,
- практическое задание.
