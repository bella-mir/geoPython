# Геоаналитика на Python

Открытое интерактивное пособие, посвящённое **основам работы с пространственными данными на Python**.

## Структура

Для работы с пособием достаточно базового понимания Python — оно подойдёт всем, кто интересуется пространственным анализом.

Материалы представлены в [Jupyter Book](https://geo-python.ru), который состоит из набора Jupyter-ноутбуков.

Все ноутбуки самодостаточны: их можно проходить как последовательно, так и выбирая отдельные интересующие темы.

Основные темы:

- [Пространственные данные](/chapters/module_1/spData_0.ipynb)
- [Проекции и системы координат](/chapters/module_2/projections_0.ipynb)
- [Пространственные операции](/chapters/module_3/geoprocessing_0.ipynb)
- [Основы сетевого анализа](/chapters/module_4/networkAnalysis_0.ipynb)
- [Растровые данные и зональная статистика](/chapters//module_5/rasters_0.ipynb)
- Создание интерактивных карт

## Курс-проект - [Boosty](https://boosty.to/bellamir)

На основе материалов пособия создан курс, который можно найти на платформе [Boosty](https://boosty.to/bellamir)

**Все модули курса связаны между собой и объединены в один проект.** К каждому модулю предусмотрено задание — часть аналитической задачи **по оценке доступности повседневных функций выбранного вами города**.

Таким образом, вы не только **освоите методы пространственного анализа на Python** и познакомитесь с библиотеками Python для работы с геоданными, но и **выполните проект**, результаты которого отобразите на интерактивной карте и опубликуете в открытом доступе (при желании).

Еженедельно выходят:

- видеоразборы занятий
- задания с рекомендуемыми шагами
- разборы библиотек Python для пространственного анализа
- другие полезные дополнительные материалы

Поддерживая проект, вы **получаете доступ к практико-ориентированному курсу по пространственному анализу** и одновременно **помогаете выпускать больше открытых образовательных материалов**.
