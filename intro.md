# Геоаналитика на Python

Интерактивное пособие, посвящённое **основам работы с пространственными данными на Python**.

На платформе [Boosty](https://boosty.to/bellamir) раз в неделю будут выходить видеоразборы занятий, а также публиковаться дополнительные полезные материалы.

## Курс-проект

Шаг за шагом вы будете **осваивать методы пространственного анализа** и знакомиться с библиотеками Python для работы с геоданными, одновременно **выполняя проект**. Для одного из районов выбранного вами города вы роведёте оценку доступности повседневных функций и отобразите результаты на интерактивной карте.

Курс состоит из модулей. Каждый модуль посвящён отдельной теме и включает задание — один из этапов итогового проекта.

## Основные темы

В рамках курса мы рассмотрим:

- Форматы пространственных данных
- Проекции и системы координат
- Пространственные операции
- Основы сетевого анализа
- Растровые данные и зональную статистику
- Создание интерактивных карт

## Формат

Курс требует лишь базового понимания Python и рассчитан на широкий круг пользователей, интересующихся пространственным анализом.

Материалы представлены в виде набора Jupyter-ноутбуков, объединённых в Jupyter Book.

Все ноутбуки самодостаточны: их можно проходить как последовательно, так и выбирая отдельные интересующие темы.
