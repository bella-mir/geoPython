# Геоаналитика на Python

Интерактивное пособие, посвящённое **основам работы с пространственными данными на Python**.

## Курс-проект

**Весь курс построен вокруг проекта - оценки доступности объектов социальной инфраструктуры для выбранного вами города.**

Шаг за шагом вы будете осваивать методы пространственного анализа и библиотеки Python для работы с геоданными, одновременно выполняя задания -- этапы общего проекта. А в конце вы создадите интерактивные карты с визуализацией резульаьлв

В результате вы оцените доступность объектов социальной инфраструктуры в выбранном районе города и создадите интерактивные карты с визуализацией результатов.

## Основные темы

В рамках курса мы рассмотрим:

- Основные типы пространственных данных
- Проекции и системы координат (CRS)
- Пространственные операции
- Основы сетевого анализа
- Растровые данные и зональную статистику
- Создание интерактивных карт

Основы Python подробно разбираться не будут. Достаточно базового понимания языка — при необходимости его легко можно получить на любом вводном онлайн-курсе.

## Формат

Курс состоит из модулей. Каждый модуль посвящён отдельной теме и включает задание — один из этапов общего проекта.

Курс состоит из набора Jupyter-ноутбуков, объединённых в Jupyter Book.

Модули можно проходить как по отдельности, так и последовательно.

Каждый модуль включает:

- краткую теорию,
- примеры кода,
- практическое задание.

На платформе [Boosty](https://boosty.to/bellamir) раз в неделю будут выходить видео-разборы занятий, а также публиковаться дополнительные полезные материалы.
